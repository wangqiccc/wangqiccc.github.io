Veröffentlicht am 2018-03-21

如何用nodejs处理excel表格数据 比如合并 排序 以及如何导入到echarts中进行处理

所需模块及方法

内部模块

  • fs
    • writeFileSync 同步写文件
    • readdirSync 同步读取文件夹
    • readFileSync 同步读取文件
  • path
    • resolve 把一个路径或路径片段的序列解析为一个绝对路径
    • join 使用平台特定的分隔符把全部给定的 path 片段连接到一起，并规范化生成的路径
  • http
    • createServer 创建一个服务器

外部模块

  • node-xlsx
    • parse 将ecxel表格转化为数据
    • build 将数据转化为excel文件
